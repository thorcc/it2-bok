## Figurer

### Rektangler

**Med fyll**

```python
pygame.draw.rect(overflate, farge, (x, y, bredde, høyde))
pygame.draw.rect(self.screen, "black", (100, 50, 125, 75))
```

![drawRect](../../../assets/bilder/pygame/drawRect.svg)

**Uten fyll**

```python
pygame.draw.rect(overflate, farge, (x, y, bredde, høyde), tykkelse)
pygame.draw.rect(self.screen, "black", (100, 50, 125, 75), 5)
```

![drawRect2](../../../assets/bilder/pygame/drawRect2.svg)

### Linjer

```python
pygame.draw.line(overflate, farge, (start_x, start_y), (slutt_x, slutt_y), tykkelse)
pygame.draw.line(self.screen, "black", (100, 50), (125, 75), 5)
```

![drawLine](../../../assets/bilder/pygame/drawLine.svg)

### Flere Figurer

[Pygame-dokumentasjoen (pygame.org/docs/ref/draw)](https://www.pygame.org/docs/ref/draw.html) viser en oversikt over alle figurene som er mulig å tegne i Pygame.
For figurer som ikke finnes i Pygame-modulen er det enklest å tegne figurene i et tegneprogram og laste inn bildefilene.

font = pygame.font.SysFont("Arial", 36)
