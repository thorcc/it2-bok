## Tabeller

En tabell i HTML bruker taggene `<table>`, `<tr>` (rad), `<th>` (overskriftscelle) og `<td>` (datacelle).

```html
<table>
    <tr>
        <th>Navn</th>
        <th>Medlemmer</th>
    </tr>
    <tr>
        <td>Skuespill</td>
        <td>18</td>
    </tr>
</table>
```

Med Jinja kan vi bruke for-løkker til å generere rader automatisk fra lister.

### Les inn fil i app.py

```python
from flask import Flask, render_template, request
import json

app = Flask(__name__)

with open("revyen.json", encoding="utf-8") as fil:
    data = json.load(fil)

# Legg inn ruter her


app.run(debug=True)
```

### Flask-rute

Les inn dataene og send dem til templaten med `render_template`:

```python
@app.get("/")
def index():
    return render_template("index.html", grupper=data)
```

### HTML-template

Bruk en Jinja-løkke til å lage én rad per gruppe:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Revyen</title>
</head>
<body>
    <h1>Revygrupper</h1>
    <table>
        <tr>
            <th>Gruppe</th>
            <th>Leder</th>
            <th>Medlemmer</th>
            <th>Klar?</th>
        </tr>
        {% for gruppe in grupper %}
        <tr>
            <td>{{ gruppe['navn'] }}</td>
            <td>{{ gruppe['leder'] }}</td>
            <td>{{ gruppe['antall_medlemmer'] }}</td>
            <td>{{ "Ja" if gruppe['klar'] == 'true' else "Nei" }}</td>
        </tr>
        {% endfor %}
    </table>
</body>
</html>
```

## Plott med Chart.js

Chart.js er et JavaScript-bibliotek for å lage interaktive diagrammer i nettleseren.
Flask beregner dataene og Chart.js tegner dem i en `<canvas>`-tag.

Last inn Chart.js i `<head>` i HTML-fila:

```html
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
```

For å sende Python-data til JavaScript bruker vi Jinja-filteret `| tojson`, som konverterer Python-lister og -ordbøker til JavaScript-format.

### Søylediagram: karakterer per fag

### Les inn fil i app.py

```python
from flask import Flask, render_template, request
import csv

app = Flask(__name__)

with open("karakterer.csv", encoding="utf-8") as fil:
    leser = csv.DictReader(fil)
    karakterer = list(leser)

# Legg inn ruter her


app.run(debug=True)
```

#### Flask-rute

Beregn data i Python og send to lister til templaten – én med etiketter og én med verdier:

```python
@app.get("/")
def index():
    fagstatistikk = {}

    for k in karakterer:
        if k["fag"] not in fagstatistikk:
            fagstatistikk[k["fag"]] = {
                antall: 1,
                total: k["karakter"]
            }
        else:
            fagstatistikk[k["fag"]]["antall"] += 1
            fagstatistikk[k["fag"]]["totalkarakter"] += 1

    fagliste = []
    snittliste = []
    for f in fagstatistikk:
        fagliste.append(f)
        snitt = fagstatistikk[f]["totalkarakter"] / fagstatistikk[f]["antall"]
        snittliste.append(snitt)


    return render_template("index.html", fagliste=fagliste, snittliste=snittliste)
```

#### HTML-template

```html
<!DOCTYPE html>
<html>
<head>
    <title>Karakterer</title>
    <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
</head>
<body>
    <h1>Gjennomsnittskarakter per fag</h1>
    <canvas id="mittDiagram"></canvas>

    <script>
        const fagliste = {{ fagliste | tojson }};
        const snitt = {{ snittliste | tojson }};

        new Chart(mittDiagram, {
            type: "bar",
            data: {
                labels: fagliste,
                datasets: [{
                    label: "Snittkarakter",
                    data: snittliste
                }]
            }
        });
    </script>
</body>
</html>
```

### Kakediagram: fordeling av revygrupper

Kakediagrammer egner seg for å vise andeler. Her viser vi størrelsen på hver revygruppe.

### Les inn fil i app.py

```python
from flask import Flask, render_template, request
import json

app = Flask(__name__)

with open("revyen.json", encoding="utf-8") as fil:
    data = json.load(fil)

# Legg inn ruter her


app.run(debug=True)
```


#### Flask-rute

Sorter gruppene etter størrelse før du henter ut kolonnene, så diagrammet viser dem i riktig rekkefølge:

```python
@app.get("/revy")
def revy():
    gruppenavn = []
    antall_medlemmer = []
    for gruppe in data:
        gruppenavn.append(gruppe["navn"])
        antall_medlemmer.append(gruppe["antall_medlemmer"])
    return render_template("revy.html", navn=gruppenavn, medlemmer=antall_medlemmer)
```

#### HTML-template

```html
<canvas id="revyDiagram"></canvas>

<script>
    const navn = {{ navn | tojson }}
    const medlemmer = {{ medlemmer | tojson }}
    new Chart(revyDiagram, {
        type: "pie",
        data: {
            labels: navn,
            datasets: [{
                data: medlemmer
            }]
        }
    });
</script>
```


## Sortering

Vi kan la brukeren velge sortering ved å sende en verdi i URL-en med `?sorter=kolonnenavn`.
Flask leser denne verdien med `request.args.get()`.

### Les inn fil i app.py

```python
from flask import Flask, render_template, request
import json

app = Flask(__name__)

with open("revyen.json", encoding="utf-8") as fil:
    data = json.load(fil)

# Legg inn ruter her


app.run(debug=True)
```

### Flask-rute

Les inn dataene og send dem til templaten med `render_template`:

```python
@app.get("/")
def index():
    sorter_på = request.args.get("sorter")
    sorterte_grupper = sorted(personer, key=lambda x: x[sorter_på], reversed=True)
    return render_template("index.html", grupper=sorterte_grupper)
```

Hvis ingen `?sorter=`-parameter er oppgitt, returnerer `request.args.get("sorter")` verdien `None`, og tabellen vises usortert.

### HTML-template

Legg til skjemaer med knapper øverst i tabellen:

```html
<form method="get">
    <input type="hidden" name="sorter" value="navn">
    <button type="submit">Sorter etter navn</button>
</form>
<form method="get">
    <input type="hidden" name="sorter" value="antall_medlemmer">
    <button type="submit">Sorter etter medlemmer</button>
</form>
<form method="get">
    <button type="submit">Nullstill</button>
</form>

<table>
    <tr>
        <th>Gruppe</th>
        <th>Leder</th>
        <th>Medlemmer</th>
        <th>Klar?</th>
    </tr>
    {% for gruppe in grupper %}
    <tr>
        <td>{{ gruppe['navn'] }}</td>
        <td>{{ gruppe['leder'] }}</td>
        <td>{{ gruppe['antall_medlemmer'] }}</td>
        <td>{{ "Ja" if gruppe['klar'] == 'true' else "Nei" }}</td>
    </tr>
    {% endfor %}
</table>
```

Hvert skjema bruker `method="get"` slik at sorteringsverdien sendes som en del av URL-en (`/?sorter=antall_medlemmer`).
Det skjulte `<input type="hidden">`-feltet bestemmer hvilken kolonne det sorteres på, og Flask leser verdien med `request.args.get("sorter")` som før.
