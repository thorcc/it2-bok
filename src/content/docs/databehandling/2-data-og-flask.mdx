Start med dette grunnoppsettet i `app.py`:

```python
from flask import Flask, render_template, request
from filbib import JSONfil, CSVfil

app = Flask(__name__)

app.run(debug=True)
```

## Tabeller

En tabell i HTML bruker taggene `<table>`, `<tr>` (rad), `<th>` (overskriftscelle) og `<td>` (datacelle).

```html
<table>
    <tr>
        <th>Navn</th>
        <th>Medlemmer</th>
    </tr>
    <tr>
        <td>Skuespill</td>
        <td>18</td>
    </tr>
</table>
```

Med Jinja kan vi bruke for-løkker til å generere rader automatisk fra lister.

### Flask-rute

Les inn dataene og send dem til templaten med `render_template`:

```python
@app.get("/")
def index():
    revy = JSONfil("revyen.json")
    return render_template("index.html", grupper=revy.data)
```

### HTML-template

Bruk en Jinja-løkke til å lage én rad per gruppe:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Revyen</title>
</head>
<body>
    <h1>Revygrupper</h1>
    <table>
        <tr>
            <th>Gruppe</th>
            <th>Leder</th>
            <th>Medlemmer</th>
            <th>Klar?</th>
        </tr>
        {% for gruppe in grupper %}
        <tr>
            <td>{{ gruppe.navn }}</td>
            <td>{{ gruppe.leder }}</td>
            <td>{{ gruppe.antall_medlemmer }}</td>
            <td>{{ "Ja" if gruppe.klar else "Nei" }}</td>
        </tr>
        {% endfor %}
    </table>
</body>
</html>
```

## Plott med Chart.js

Chart.js er et JavaScript-bibliotek for å lage interaktive diagrammer i nettleseren.
Flask beregner dataene, og Chart.js tegner dem i en `<canvas>`-tag.

Last inn Chart.js i `<head>`:

```html
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
```

For å sende Python-data til JavaScript bruker vi Jinja-filteret `| tojson`, som konverterer Python-lister og -ordbøker til JavaScript-format.

### Søylediagram – karakterer per fag

#### Flask-rute

Beregn data i Python og send to lister til templaten – én med etiketter og én med verdier:

```python
@app.get("/")
def index():
    karakterer = CSVfil("karakterer.csv")

    fag = ["Matematikk", "Norsk", "IT2"]
    snitt = []
    for f in fag:
        snitt.append(karakterer.gjennomsnitt_med_filter("fag", f, "karakter"))

    return render_template("index.html", fag=fag, snitt=snitt)
```

#### HTML-template

```html
<!DOCTYPE html>
<html>
<head>
    <title>Karakterer</title>
    <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
</head>
<body>
    <h1>Gjennomsnittskarakter per fag</h1>
    <canvas id="mittDiagram"></canvas>

    <script>
        const fag = {{ fag | tojson }};
        const snitt = {{ snitt | tojson }};

        new Chart(mittDiagram, {
            type: "bar",
            data: {
                labels: fag,
                datasets: [{
                    label: "Snittkarakter",
                    data: snitt
                }]
            }
        });
    </script>
</body>
</html>
```

### Kakediagram – fordeling av revygrupper

Kakediagrammer egner seg for å vise andeler. Her viser vi størrelsen på hver revygruppe.

#### Flask-rute

Sorter gruppene etter størrelse før du henter ut kolonnene, så diagrammet viser dem i riktig rekkefølge:

```python
@app.get("/revy")
def revy():
    revy = JSONfil("revyen.json")
    revy.sorter("antall_medlemmer", synkende=True)
    navn = revy.kolonne("navn")
    medlemmer = revy.kolonne("antall_medlemmer")
    return render_template("revy.html", navn=navn, medlemmer=medlemmer)
```

#### HTML-template

```html
<canvas id="revyDiagram"></canvas>

<script>
    new Chart(revyDiagram, {
        type: "pie",
        data: {
            labels: {{ navn | tojson }},
            datasets: [{
                data: {{ medlemmer | tojson }}
            }]
        }
    });
</script>
```

## Sortering

Vi kan la brukeren velge sortering ved å sende en verdi i URL-en med `?sorter=kolonnenavn`.
Flask leser denne verdien med `request.args.get()`.

### Flask-rute

```python
@app.get("/")
def index():
    revy = JSONfil("revyen.json")
    sorter_på = request.args.get("sorter")
    if sorter_på:
        revy.sorter(sorter_på, synkende=True)
    return render_template("index.html", grupper=revy.data)
```

Hvis ingen `?sorter=`-parameter er oppgitt, returnerer `request.args.get("sorter")` verdien `None`, og tabellen vises usortert.

### HTML-template

Legg til skjemaer med knapper øverst i tabellen:

```html
<form method="get">
    <input type="hidden" name="sorter" value="navn">
    <button type="submit">Sorter etter navn</button>
</form>
<form method="get">
    <input type="hidden" name="sorter" value="antall_medlemmer">
    <button type="submit">Sorter etter medlemmer</button>
</form>
<form method="get">
    <button type="submit">Nullstill</button>
</form>

<table>
    <tr>
        <th>Gruppe</th>
        <th>Leder</th>
        <th>Medlemmer</th>
        <th>Klar?</th>
    </tr>
    {% for gruppe in grupper %}
    <tr>
        <td>{{ gruppe.navn }}</td>
        <td>{{ gruppe.leder }}</td>
        <td>{{ gruppe.antall_medlemmer }}</td>
        <td>{{ "Ja" if gruppe.klar else "Nei" }}</td>
    </tr>
    {% endfor %}
</table>
```

Hvert skjema bruker `method="get"` slik at sorteringsverdien sendes som en del av URL-en (`/?sorter=antall_medlemmer`).
Det skjulte `<input type="hidden">`-feltet bestemmer hvilken kolonne det sorteres på, og Flask leser verdien med `request.args.get("sorter")` som før.
